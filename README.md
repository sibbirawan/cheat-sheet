## ⚙️ Usage

The cheat-sheet consists of copy-paste ready commands. When using these commands, ensure you adjust the VMIDs, hostnames, and paths to fit your specific environment.

### Example Usage

1. Locate the command you need from the Proxmox VE guide.
2. Copy the command.
3. Open your command line interface.
4. Paste the command.
5. Modify VMIDs or hostnames as necessary.
6. Press Enter to execute the command.
